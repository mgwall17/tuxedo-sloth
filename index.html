What is Exosphere?

Exosphere makes it easy to use your OpenStack allocations with a user-friendly interface designed with researchers and non-IT professionals in mind.

Exosphere makes it easy to deploy code and run services on OpenStack clouds with a user-friendly interface. Designed for researchers, it bridges the gap between complex admin tools and intuitive proprietary services. Customize your experience with branding and single sign-on integration, no advanced IT knowledge needed.

Frequently Asked Questions

OpenStack is the operating system and APIs that power public research clouds like Jetstream2 and Chameleon, private clouds at organizations like Wikimedia and CERN, and public commercial clouds like OVH, Fuga, Vexxhost, and Leafcloud. Any organization can run OpenStack on its own hardware to provide a cloud service for its own community, or for the world.

  • Run graphics-intensive software like Blender and Metashape on powerful GPUs.

  • Run large language models in your own private, customizable environment with Ollama.

  • Develop and serve a rich web application with a database server.

  • Work with bleeding-edge software projects without possibly breaking your own computer.

  • Teach a computational workshop that delivers a customized, easy-to-access sandbox environment for student exercises.

  • 100% self-hostable, 99% standalone client application

  • Two small proxy servers secure web browser connections to OpenStack APIs, and interactive services running on cloud instances.

Exosphere is free, but development work is powered by community contributions via Open Collective, so folks are welcome donate there to support this project.
